# Change Log

All notable changes to the "HyperType" extension will be documented in this file.

## [0.0.1] - 2025-11-08

### Added
- Initial release of HyperType
- Pixel font rendering using M6X11 Plus font
- Floating character animations with three directional modes:
  - Upper-left drift for regular typing
  - Upper-right drift for backspace
  - Straight-up drift for Enter and Tab
- 74-color gradient system for regular characters
- Special color system for SPACE, ENTER, TAB, BACKSPACE, DELETE, and SHIFT keys
- Expanding corner bracket animations around typed characters
- Screen shake effects on keypresses
- `>>>` marker in gutter when pressing Enter
- Capitalized text display for all keystrokes
- HyperType sidebar panel showing recent keystrokes
- No fade animations - characters maintain full opacity
- 150ms hold time at final position before disappearing
- No corner brackets for ENTER, TAB, and BACKSPACE keys
